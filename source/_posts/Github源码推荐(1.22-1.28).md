## 三、习惯了`OC`下的`YYModel`等`Dictionary`转`Model`工具，在`Swift`下是否不适应呢？试试`SwiftyJSON`吧，一定会为你带来更好的体验。


### 热度
```
Star: 16000+
```

### CocoaPods
```
pod 'SwiftyJSON'
```

### Carthage
```
github "SwiftyJSON/SwiftyJSON"
```

### `Github`主页
[https://github.com/SwiftyJSON/SwiftyJSON](https://github.com/SwiftyJSON/SwiftyJSON)
